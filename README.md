## 🚌 **BANGKOK BUS & CAR NAVIGATION - ဘန်ကောက်ကားလမ်းညွှန်**

### ✅ **အဓိကအင်္ဂါများ**

#### 🚆 **BTS (ဘီတီအက်စ်)**
- 📍 အင်္ဂလိပ်စကား လမ်းကြေး
- ⭐ လမ်းကြေးစီစဥ်
- 💰 ခရေးခွင့်အချက်အလက်
- 🕐 လုပ်ငန်းအချိန်
- ✓ အမြန်သယ်ယူမှု

#### 🚇 **MRT (မြေအောက်ရထား)**
- 📍 လမ်းကြေးများ
- ⭐ အဆင်ပြေ ရထား
- 💰 ခရေးခွင့်အချက်အလက်
- 🕐 လုပ်ငန်းအချိန်

#### 🚌 **ဘက်စ်ကားများ**
- 📍 ဘက်စ်နံပါတ်အရေး
- ⭐ အများအသုံးပြုလုပ်သည့် လမ်းကြေး
- 💰 အကုန်အလျော့
- 🕐 သတ်မှတ်အချိန်

#### 🚕 **Taxi**
- 📍 ခေါ်နည်း
- ⭐ အီမီတာစနစ်
- 💰 ခွဲခြားခြင်း
- 🕐 ၂၄/၇ အဖွင့်

#### 🚗 **Grab**
- 📍 အပ်ပ်ခေါ်နည်း
- ⭐ ဒီဂျီတယ်စနစ်
- 💰 အီမီတာစနစ်
- 🕐 ၂၄/၇ အဖွင့်

#### 🗺️ **လမ်းကြေးများ**
- 📍 အများဆုံးအရေးအမိန်လမ်းကြေး
- ⭐ သယ်ယူမှုနည်း
- 💰 အကုန်အစရိုး
- 🕐 အချိန်

### 📊 **သယ်ယူမှုအုံးမုံခြင်း**

| သယ်ယူမှုနည်း | စတင်နှုန်း | နောက်ဆက်တွဲ | အချိန်အခြေခံ | ၂၄/၇ |
|---------------|------------|------------|----------|------|

### 💡 **အကျိုးအကြေး**

#### ✓ **အကျိုးများ**
- ✓ BTS/MRT - အမြန်ဆုံးသည့် သယ်ယူမှု
- ✓ ဘက်စ် - အကုန်အလျော့ သည့်သယ်ယူမှု
- ✓ Taxi - အယုံကြည်ရသည့် သယ်ယူမှု
- ✓ Grab - အဆင်ပြေဆုံးသည့် သယ်ယူမှု
- ✓ အွန်လိုင်းမဲ့ အချက်အလက်များ

#### ⚠️ **သတိအိုင်များ**
- ⚠️ အိမ်နီးကပ်သည့်အခါ Taxi မသုံးပါစို့
- ⚠️ ည၏အချိန်တွင် အချီလူတွေ ကိုယ်ဂီယာရှိလည်းရှိ
- ⚠️ အုံးမုံခြင်းတွင် ခြင်းစီစီလုံးရုံ အားလုံးအတွင်း မီတာ ဖွင့်ထားခြင်း
